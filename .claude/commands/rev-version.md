# Rev Version Command

Bump the version of todd-lab to $ARGUMENTS.

## Instructions

1. Update the version in `.todd/lab/client-nextjs/package.json` to the specified version
2. Stage and commit with message: `chore: bump version to <version>`
3. Report the change

## Version Format

- Use semantic versioning: `MAJOR.MINOR.PATCH` (e.g., `1.0.3`, `2.1.0`)
- If no version specified, increment PATCH by 1

## Package Location

`.todd/lab/client-nextjs/package.json`
