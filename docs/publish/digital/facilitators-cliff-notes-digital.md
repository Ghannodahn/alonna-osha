## Session Flow

| Time | Section | Key Points |
|------|---------|------------|
| **0-1 min** | OPENING | "I'm [Name], been doing this [X years]..." • 4 things • 15 minutes |
| **1-3 min** | LIFTING | "Your back doesn't heal like a broken board" • Get close • Test weight • Ask for help • Team lifts |
| **3-5 min** | HEIGHTS | "3 points touching, someone knows you're up there" • 4:1 ratio • 3-point contact • Spotter • Top step = handhold |
| **5-7 min** | AWARENESS | "Call out before you move" • "Heads up!" • Know neighbors • Staging • Shared space |
| **7-9 min** | TOOLS | "PPE on, guards on, if it feels wrong—stop" • Glasses • Gloves • Guards • Cords • "Stop and ask" |
| **9-12 min** | DEMO | Ladder setup OR Lifting OR Calls • You do → They try → Group confirms |
| **12-15 min** | CLOSE | "Big four: lift smart, 3 points, watch out, respect tools" • "Unsure? Ask." • Point to leads |

---

## Quick Engagement Questions

| Point | Ask |
|-------|-----|
| Lifting | "Anyone lifted something they shouldn't have?" |
| Heights | "Show me 4:1" (if ladder nearby) |
| Awareness | "What's your call when moving something long?" |
| Tools | "What do you check before using a tool?" |

---

## Quick Saves

| Situation | Response |
|-----------|----------|
| **Don't know answer** | "Let me find out and get back to you" |
| **Quiet group** | "Turn to your neighbor and discuss..." |
| **Dominator** | "Thanks—who else?" + break eye contact |
| **Skeptic** | "Your experience could help newer folks" |
| **Nervous person** | Use RACE (below) |

---

## RACE (for nervous participants)

| | |
|---|---|
| **R**ecognize | Notice tension (white knuckles, not moving) |
| **A**cknowledge | "That wobbly feeling is totally normal" |
| **C**oncrete | Give one specific action |
| **E**mpower | "Good—you've got it" |

## Messages to Remember

> **Lifting:** "Your back doesn't heal like a broken board."

> **Heights:** "Three points touching. Someone knows you're up there."

> **Awareness:** "Call out before you move."

> **Tools:** "If it feels wrong, stop."
