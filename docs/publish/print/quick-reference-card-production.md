# Quick Reference Card Production Guide

**Document:** Two-Sided Pocket Reference Card for Construction Safety Facilitators
**Version:** 1.0
**Card Dimensions:** 4" × 6" (index card standard)

---

## Card Overview

The quick reference card is a two-sided laminated pocket card that facilitators can carry on the jobsite. It provides at-a-glance access to:

- **Front Side:** "When Someone's Struggling" - Phrase comparisons and RACE framework
- **Back Side:** "15-Minute Session Flow" - Timeline with key points

## Front Side Content

### Title: "When Someone's Struggling"
**Subtitle:** What to say (and what to skip)

### Phrase Comparisons (5 rows)

| ❌ Skip | → | ✓ Say Instead |
|--------|---|---------------|
| "Don't fall" | → | "Three points on the ladder" |
| "Calm down" | → | "I'm right here" |
| "It's not a big deal" | → | "That's a reasonable concern" |
| "Just do it" | → | "One step at a time" |
| "Be careful" | → | "Watch the nail by your foot" |

### RACE Framework Box

**Title:** RACE — 10 seconds when someone freezes

| Letter | Action | Example |
|--------|--------|---------|
| **R** | Recognize their state | (white knuckles, not moving) |
| **A** | Acknowledge briefly | ("Ladders feel wobbly at first.") |
| **C** | Concrete action — one specific move | ("Right hand to the next rung.") |
| **E** | Empower | ("Good. You've got it.") |

## Back Side Content

### Title: "15-Minute Session Flow"

### Vertical Timeline (7 stops)

| Time | Section | Key Points |
|------|---------|------------|
| 0-1 min | **OPENING** | "I've done this a while..." • Today: 4 points, 15 minutes |
| 1-3 min | **LIFTING & CARRYING** | Get close, test weight, ask for help |
| 3-5 min | **WORKING AT HEIGHTS** | 4:1 ratio, 3-point contact, spotter |
| 5-7 min | **AWARENESS** | "Heads up!", Know your neighbors |
| 7-9 min | **TOOLS** | PPE on, Guards stay on, "Feels wrong? Stop and ask" |
| 9-12 min | **DEMO** | You do it, They try, Group confirms |
| 12-15 min | **CLOSE** | "If unsure, ask." Point to site lead. |

### Footer
**Text:** Your job: prime awareness. Details come at task.

## Printing Instructions

### For Professional Print Shop

1. **Submit file:** `facilitator-card-2.png` (or preferred variant)
2. **Specify:**
   - Quantity: [Per order]
   - Size: 4" × 6" index card
   - Stock: 80-100 lb cover (216-270 gsm)
   - Sides: 2-sided, full color
   - Finish: Matte or gloss lamination (5-7 mil recommended)
   - Cut: Standard trim, optional rounded corners (0.125" radius)

3. **Proofing:**
   - Request a physical proof before full run
   - Verify text is legible at actual size
   - Check color accuracy against hex values

### For Office/DIY Printing

1. **Paper:** Cardstock (65-110 lb / 176-300 gsm)
2. **Printer settings:**
   - Paper size: 4×6 or print on letter and trim
   - Quality: Best/Photo
   - Color: Full color
3. **Lamination:** Use self-seal or pouch laminator (5 mil minimum)
4. **Cutting:** Paper trimmer or scissors with template

### Lamination Specifications

| Specification | Recommendation |
|--------------|----------------|
| Pouch thickness | 5-7 mil (0.005"-0.007") |
| Type | Matte finish (reduces glare outdoors) |
| Edge seal | At least 0.25" past card edge |
| Optional | Corner punch for lanyard hole |

## Quality Checklist

### Before Production

- [ ] Text is legible at 4×6" size (minimum 7pt body text)
- [ ] High contrast for outdoor readability
- [ ] CMYK color values verified
- [ ] No spelling errors
- [ ] Bleed area present if full-bleed design

### After Production

- [ ] Card fits in standard pocket
- [ ] Both sides readable in direct sunlight
- [ ] Lamination sealed properly (no peeling edges)
- [ ] Colors match design specifications
- [ ] Card survives handling test (bend, dust, light moisture)

## Accessory Options

### Lanyard Attachment

For hands-free carrying:
1. Punch 0.25" hole in corner (use rounded corner punch)
2. Position hole 0.375" from edges
3. Use badge lanyard clip or split ring

### Badge Holder

Alternative to lamination:
- Use 4×6 horizontal badge holder
- Allows card replacement without new lamination
- Less durable for dirty jobsite conditions

## Reorder Information

### When to Reorder

- Cards show wear (cracked lamination, faded text)
- Content updates required
- New facilitators onboarded
