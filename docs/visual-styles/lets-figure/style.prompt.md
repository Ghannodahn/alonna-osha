# Let's Figure This Out Visual Style - Image Generation Prompt

## Style: "The Whiteboard Session"

Create a safety infographic with these exact visual specifications:

### Visual Tone
Collaborative, exploratory, educational. Like a productive training session on a whiteboard. Think marker diagrams, flowcharts, thinking out loud together. Inquiry-driven learning.

### Color Palette (EXACT HEX CODES - Whiteboard Marker Colors)
- Primary text/diagrams: Marker Blue #2C5282
- Background: Whiteboard White #FFFFFF (clean, bright)
- Questions/inquiry: Marker Green #38A169
- Answers/conclusions: Marker Blue #3182CE
- Warnings/alerts: Marker Red #E53E3E
- Supporting text: Marker Black #2D3748

### Typography Style
- Headers: Marker-style font (like Permanent Marker or Caveat Bold)
- Body: Clean, friendly sans-serif (like Open Sans or Lato)
- Questions marked with "?" prominently
- Answers/resolutions marked with "✓"

### Layout Requirements
- Diagram-centric - flowcharts, process flows, connected thinking
- Question at top driving the content
- Cause → Effect → Solution mapping
- Hand-drawn boxes, circles, arrows connecting concepts
- Before/after or comparison layouts welcome

### Graphic Elements
- Flowchart elements: boxes, arrows, diamonds (decision points)
- Hand-drawn quality to shapes (not perfectly geometric)
- Marker texture on lines and fills
- Connection lines showing relationships
- "?" and "✓" as visual cues throughout

### Whiteboard Effects
- Clean white background
- Marker imperfections (slight texture, occasional overlap)
- NOT cluttered - whiteboard has space
- Could show slight marker ghosting from previous content
- Diagrams feel sketched during a meeting

### DON'T Include
- Polished, corporate graphics
- Photo-realistic imagery
- Dark or heavy backgrounds
- Static lists without connections
- Formal, institutional aesthetic

### MUST Include
- A question (?) that drives the content
- A diagram or flowchart showing relationships
- Cause → effect → solution visual flow
- Resolution marker (✓) for conclusions
- Collaborative, "we figured this out together" energy
