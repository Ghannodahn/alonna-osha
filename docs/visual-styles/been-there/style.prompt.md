# Been There Visual Style - Image Generation Prompt

## Style: "The Field Notebook"

Create a safety infographic with these exact visual specifications:

### Visual Tone
Warm, personal, experiential. Like wisdom passed down from a veteran. Think well-used field notebook with margin notes, trusted mentor's journal, worn but reliable.

### Color Palette (EXACT HEX CODES)
- Primary text: Dark Brown #3D2B1F
- Background: Cream/Aged Paper #F5F0E8
- Accent: Workworn Orange #CC7722 (warm, not bright)
- Secondary: Sage Green #8B9A6B (for tips and positive notes)
- Highlight: Faded Yellow #F7E7A2 (for emphasized passages)

### Typography Style
- Headers: Rounded, friendly sans-serif (like Nunito or Varela Round)
- Body: Readable serif (like Georgia or Merriweather)
- Mixed case, conversational capitalization
- Slightly informal, approachable

- Organic flow, not rigidly gridded
- Story/anecdote callout boxes with dashed or notebook-style borders

- "Wisdom box" sections with rounded corners
- Comfortable margins, breathing room

### Texture (Subtle)

- NOT distracting - texture enhances, doesn't dominate

### DON'T Include
- Sharp, industrial edges
- Bright, saturated colors
- Technical/blueprint aesthetic
- Perfect geometric shapes
- Cold, clinical feel

### MUST Include

- Warm, human, approachable aesthetic throughout
