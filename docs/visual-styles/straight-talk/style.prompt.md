# Straight Talk Visual Style - Image Generation Prompt

## Style: "The Engineer's Clipboard"

Create a safety infographic with these exact visual specifications:

### Visual Tone
Industrial, precise, no-nonsense, technical documentation aesthetic. Think safety data sheet meets field guide. Clean and efficient, not decorative.

### Color Palette (EXACT HEX CODES)
- Primary text: Near black #1A1A1A
- Background: Off-white #FAFAFA
- Accent: Safety Yellow #FFD100 (use sparingly for critical callouts only)
- Secondary: Industrial Gray #4A4A4A
- Alert (danger only): ANSI Red #C41E3A

### Typography Style
- Headers: Bold, condensed, industrial sans-serif (like Impact or Oswald)
- Body: Clean neutral sans-serif (like Helvetica or Arial)
- ALL CAPS for action items and headers
- Strong visual hierarchy with clear size differences

### Layout Requirements
- Grid-based, structured, highly organized
- Generous whitespace - minimum 30%
- Single-column clarity preferred
- Horizontal rule dividers between sections
- Bold header bar at top (black or dark gray with white text)

### Graphic Elements
- Filled square bullets (■) not circles
- Minimal iconography - functional only, no decoration
- Icons must be geometric, single-weight line style
- No gradients, shadows, or 3D effects - completely flat design
- No decorative borders or flourishes

### DON'T Include
- Rounded corners (use sharp corners)
- Soft colors or pastels
- Illustrations with personality
- Emoji or casual markers
- Decorative backgrounds or textures
- Multiple columns or complex layouts

### MUST Include
- Clear visual hierarchy (what to read first/second/third is obvious)
- Bold, scannable structure
- Warning symbol (⚠) only if showing danger content
- Professional, industrial aesthetic throughout
