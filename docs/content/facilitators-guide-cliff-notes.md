# Cliff Notes: 15-Minute Safety Briefing

**Glance-at-during-session reference. One page. All you need.**

---

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  0-1 min   OPENING                                                          │
│            "I'm [Name], been doing this [X years]..."                       │
│            "4 things: lift, ladder, awareness, tools"                       │
│            "15 minutes, then we build"                                      │
├─────────────────────────────────────────────────────────────────────────────┤
│  1-3 min   LIFTING                                                          │
│            Message: "Your back doesn't heal like a broken board"            │
│            → Get close → Test weight → Ask for help → Team lifts            │
│            Story: [yours]                                                   │
│            Ask: "Anyone lifted something they shouldn't have?"              │
├─────────────────────────────────────────────────────────────────────────────┤
│  3-5 min   HEIGHTS                                                          │
│            Message: "3 points touching, someone knows you're up there"      │
│            → 4:1 ratio → 3-point contact → Spotter → Top step = handhold    │
│            Story: [yours]                                                   │
│            Ask: "Show me 4:1" (if ladder nearby)                            │
├─────────────────────────────────────────────────────────────────────────────┤
│  5-7 min   AWARENESS                                                        │
│            Message: "Call out before you move"                              │
│            → "Heads up!" → Know neighbors → Staging → Shared space          │
│            Story: [yours]                                                   │
│            Ask: "What's your call when moving something long?"              │
├─────────────────────────────────────────────────────────────────────────────┤
│  7-9 min   TOOLS                                                            │
│            Message: "PPE on, guards on, if it feels wrong—stop"             │
│            → Glasses → Gloves → Guards → Cords → "Stop and ask"             │
│            Story: [yours]                                                   │
│            Ask: "What do you check before using a tool?"                    │
├─────────────────────────────────────────────────────────────────────────────┤
│  9-12 min  DEMO (pick one)                                                  │
│            [ ] Ladder: 4:1 setup → 3-point climb → spotter position         │
│            [ ] Lifting: close-squat-legs → volunteer tries                  │
│            [ ] Calls: "Heads up!" → group practices together                │
│            Pattern: You do → They try → Group confirms                      │
├─────────────────────────────────────────────────────────────────────────────┤
│  12-15 min CLOSE                                                            │
│            "Big four: lift smart, 3 points, watch out, respect tools"       │
│            "Unsure? Ask. That's smart, not slow."                           │
│            "Go-to people: [point to leads]"                                 │
│            "Let's build smart and look out for each other."                 │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Quick Saves

| Situation | Response |
|-----------|----------|
| **Don't know answer** | "Let me find out and get back to you" |
| **Quiet group** | "Turn to your neighbor and discuss..." |
| **Dominator** | "Thanks—who else?" + break eye contact |
| **Skeptic** | "Your experience could help newer folks" |
| **Nervous person** | Acknowledge → One concrete step → "You've got it" |

---

## RACE (for nervous participants)

**R**ecognize (notice tension) → **A**cknowledge ("totally normal") → **C**oncrete (one action) → **E**mpower ("you've got it")
