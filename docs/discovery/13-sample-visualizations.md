# Sample Visualizations: Ladder Safety in All Five Styles

## Purpose

This document demonstrates how the same safety content (ladder safety) would be visualized in each of the five conversation styles. Each sample shows:

1. A complete content block as it might appear
2. ASCII-art representation of the visual layout
3. Notes on key visual elements

**Common content being visualized:**
- 3-point contact rule
- Spotter requirement
- Don't overreach
- Top step isn't a step

## Testing Questions

For each visualization, consider:

1. **Distinctiveness**: Could someone correctly identify the style without labels?
2. **Appropriateness**: Does the visual feel consistent with the written tone?
3. **Effectiveness**: Does the design enhance comprehension?
4. **Feasibility**: Can this be produced with available tools/skills?
5. **Scalability**: Does it work at different sizes/formats?
